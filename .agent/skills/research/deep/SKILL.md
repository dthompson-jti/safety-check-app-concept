# Research Deep

Conduct deep research on a topic.

## When to Use
- Need comprehensive understanding
- Exploring new technologies
- Investigating best practices

## Approach

### Phase 1: Define Research Questions
- What do we need to learn?
- What are the key unknowns?

### Phase 2: Information Gathering
- Search web for authoritative sources
- Review documentation
- Find case studies

### Phase 3: Synthesis
- Extract key findings
- Note conflicting information
- Identify patterns

### Phase 4: Recommendations
- Summarize learnings
- Note implications for project
- Identify next steps

## Output
Research notes with sources cited.
